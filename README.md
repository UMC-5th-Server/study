# 🖤Study🖤
UMC 5th Node.js 스터디

## 1️⃣ Rules
- 본인 닉네임으로 branch 생성 후 본인 브랜치에만 커밋
- main으로 PR 🚫 > 스터디 종료 후 main에 일괄적으로 합칠 예정!
- node_modules 파일은 .gitignore에 추가
  
## 2️⃣ Commit Message Convention
> 커밋태그: 내용 # branch명
ex. `feat : 2주차 과제 업로드 #jay'

| command | mean |
| --- | --- |
| feat | 과제 업로드 |
| fix | 버그 수정 |
| docs | 문서 수정 |
| style | 코드 포맷 변경 |
| test | 테스트 코드 추가 |
| refactoring | 코드 리팩토링 |
| chore | 빌드, 패키지 매니저 수정 |
